# Book-Nook-Sisters

A book tracking ios app that helps you keep track of what you read and how you feel while reading. You can add notes with your voice, photos, or sketches, and get quick AI-generated summaries of your books. It shows your reading habits with clear, visual stats and lets you connect with others through a book swap feature. You can use it offline and it will sync your progress when you’re back online. It also rewards you with achievements to keep you motivated.

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
